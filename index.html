Projects

Turning ideas into impactful solutions—one project at a time.

Tic-Tac-Toe AI - Q-Learning Implementation

Tic-Tac-Toe AI is a reinforcement learning project that implements the Q-Learning algorithm to train an agent to play optimally. The AI learns by exploring different board states, updating Q-values based on rewards, and improving its strategy over thousands of self-play games.

Built with Python, NumPy, and Streamlit, the project features a Q-table-based learning system, persistent training storage using pickle, and an interactive web interface for users to challenge the trained AI.
